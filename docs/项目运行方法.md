# 新闻头条项目运行方法

## 项目概述

新闻头条管理系统是一个基于Spring Boot 3.0 + Vue 3的现代化Web应用，采用多数据源架构设计，支持高并发访问和海量内容存储。

### 技术栈

- **后端**: Spring Boot 3.0.2 + Java 17
- **前端**: Vue 3 + Element Plus + Vite
- **数据库**: MySQL 8.0 + MongoDB 5.0
- **对象存储**: MinIO
- **缓存**: Redis (规划中)
- **构建工具**: Maven 3.6+ + Node.js 20.19.0+

### 2.4 MinIO安装配置

```bash

# 创建数据目录
```

## 8. Docker部署（可选）

### 8.1 Docker Compose配置

创建 `docker-compose.yml` 文件：

```yaml
version: '3.8'

services:
  mysql:
    image: mysql:8.0
    container_name: news-mysql
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: news_db
      MYSQL_USER: news_user
      MYSQL_PASSWORD: news_password
    ports:
      - "3306:3306"
    volumes:
      - mysql_data:/var/lib/mysql
      - ./init.sql:/docker-entrypoint-initdb.d/init.sql
    networks:
      - news-network

  mongodb:
    image: mongo:5.0
    container_name: news-mongodb
    environment:
      MONGO_INITDB_ROOT_USERNAME: admin
      MONGO_INITDB_ROOT_PASSWORD: admin123
      MONGO_INITDB_DATABASE: News_MongoDB
    ports:
      - "27017:27017"
    volumes:
      - mongodb_data:/data/db
    networks:
      - news-network

  redis:
    image: redis:7-alpine
    container_name: news-redis
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data
    networks:
      - news-network

  minio:
    image: minio/minio:latest
    container_name: news-minio
    environment:
      MINIO_ROOT_USER: minioadmin
      MINIO_ROOT_PASSWORD: minioadmin
    ports:
      - "9000:9000"
      - "9001:9001"
    volumes:
      - minio_data:/data
    command: server /data --console-address ":9001"
    networks:
      - news-network

  backend:
    build:
      context: ./News_SpringBoot
      dockerfile: Dockerfile
    container_name: news-backend
    ports:
      - "8080:8080"
    environment:
      SPRING_DATASOURCE_URL: jdbc:mysql://mysql:3306/news_db
      SPRING_DATASOURCE_USERNAME: news_user
      SPRING_DATASOURCE_PASSWORD: news_password
      SPRING_DATA_MONGODB_URI: mongodb://admin:admin123@mongodb:27017/News_MongoDB
      CUSTOM_MINIO_ENDPOINT: http://minio:9000
    depends_on:
      - mysql
      - mongodb
      - minio
    networks:
      - news-network

  frontend:
    build:
      context: ./News_Vue
      dockerfile: Dockerfile
    container_name: news-frontend
    ports:
      - "5173:5173"
    depends_on:
      - backend
    networks:
      - news-network

volumes:
  mysql_data:
  mongodb_data:
  redis_data:
  minio_data:

networks:
  news-network:
    driver: bridge
```
